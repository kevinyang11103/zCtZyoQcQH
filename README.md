# 个人记账管理系统 python178

## 项目概述

个人记账管理系统是一款帮助用户高效管理个人财务的应用。系统基于Python Django框架开发，具备用户登录、个人信息管理、收支记录及统计等功能。

## 技术栈

- **后端：** Python Django
- **数据库：** MySQL
- **前端：** HTML

## 功能模块

- **用户登录：** 用户可登录系统进行操作。
- **个人信息管理：** 用户可修改个人信息。
- **收入记录：** 记录收入流水，支持收入记账管理。
- **支出记录：** 记录支出流水，支持支出记账管理。
- **统计功能：** 提供收入和支出的统计。

## 系统角色

- **普通用户：** 可进行登录、个人信息修改、收支记录及查询等操作。

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 环境配置：确保Python、Django和MySQL环境正确安装。
2. 数据库设置：创建相应数据库和表结构。
3. 程序部署：运行Django项目。
